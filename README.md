
# Study Battle — Web Quiz Game

A lightweight, web-based multiple-choice quiz game with competitive leaderboards and **real-time Match mode**. Use it to study solo or compete with classmates.

## Features
- Modes: **Practice**, **Timed**, **Challenge** (seeded order), **Match (real-time)**
- Load questions from a simple **JSON** file
- Instant feedback & optional explanations
- Score, accuracy, and time tracking
- Leaderboards by **class/group code**
- Works offline (local leaderboard) or online (Firebase Realtime Database)

## Quick Start (GitHub Pages)
1. Create a new **public** GitHub repo and upload these files.
2. Go to **Settings → Pages**.
3. **Source:** `main` branch, **Folder:** `/ (root)` → **Save**.
